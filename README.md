# Sign Up Form Example
A signup form created with the guidance of
[The Odin Project](https://www.theodinproject.com/lessons/node-path-intermediate-html-and-css-sign-up-form).

Here's the [preview](https://ernestviola.github.io/sign-up-form/)

## Lessons
 - Create and style a form
 - Simple form validations
 - Using downloaded web fonts
